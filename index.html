★ CAPSTONE PROJECT
ALCOHOL MARKER
PRODUCTION LINE
◈ SRH Advanced Programming · June 2026

A full-stack simulation of a four-stage alcohol marker manufacturing line. A Python/Flask backend runs the finite-state machine (IDLE → RUNNING → FAULTED), writing every cycle to InfluxDB in real time. An HTML HMI gives operators Start / Stop / Reset controls and live fault display. A Grafana dashboard shows machine state, part counts, ethanol concentration, and temperature — all auto-provisioned, no manual setup required.

Python · Flask InfluxDB Grafana Docker Compose
